# Trader Performance vs Market Sentiment Analysis

Analysis of the relationship between Bitcoin Fear & Greed Index 
and trader performance on Hyperliquid.

## Key Findings
- Extreme Fear = highest avg PnL per trade (contrarian opportunity)
- Extreme Greed = most trade volume but lowest win rates
- Top traders size up during Fear; bottom traders retreat
- Long positions outperform during Fear; Shorts outperform during Greed

## Dataset
- Bitcoin Fear & Greed Index (2018–2025)
- Hyperliquid Historical Trader Data (211K+ trades)
